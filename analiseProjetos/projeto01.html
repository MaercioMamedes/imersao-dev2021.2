O código

O projeto 01 - Calculadora de média, tem a finalidade de calcular a média aritmética de quatro notas e imprimir na tela o resultado. Inicialmente a execução desse projeto tem o objetivo de trabalhar os conceitos iniciais de das tecnologias utilizadas na para web, focando principalmente na linguagem JavaScript e como ela interage com o documento HTML.

Dentro do universo das tecnologias da web, temos diversasa linguagens capzes de atuar nesta área, entretanto tudo, que é gerado dentro do servidor e enviado para o cliente, é resumido num documento HTML, cuja sua formatação e estilização é codificado pela linguagem CSS e toda interatividade dinamismo da página é definido pelos códigos de JavaScript.
